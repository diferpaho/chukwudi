## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Enter your email and password

## Usage

After logging in, you will be taken to the home page where you can add products to the shopping cart by clicking the "Add" button.
